ACCESIBILIDAD.

La página web de escarcega.tecnm.mx ha sido desarrollada utilizando tecnologías modernas como Angular para la estructuración del contenido dinámico y CSS3 con Bootstrap para el diseño responsivo, cumpliendo con las pautas de accesibilidad definidas por el W3C.

El objetivo de estas directrices es garantizar el acceso a la información digital a todo tipo de usuarios, proporcionando una experiencia clara, accesible y adaptable a distintos dispositivos.

Este sitio web ha sido validado mediante herramientas de evaluación de accesibilidad como TAW, y probado en diferentes entornos y dispositivos, alcanzando un nivel de cumplimiento conforme a estándares internacionales de accesibilidad web.

Entre las medidas adoptadas para lograr estos objetivos se encuentran las siguientes:

Diseño Responsivo: El sitio se adapta a distintos tamaños de pantalla gracias al uso de Bootstrap, permitiendo una correcta visualización en dispositivos móviles, tabletas y equipos de escritorio.

Hojas de Estilo: Se emplea CSS3 para estructurar y presentar el contenido de manera visual, garantizando consistencia, claridad y compatibilidad entre navegadores modernos.

Usabilidad y Navegación: La navegación del sitio está organizada de forma estructurada, intuitiva y coherente, facilitando el acceso a la información mediante menús claros y jerarquías bien definidas.

Compatibilidad: El sistema ha sido desarrollado para funcionar correctamente en los principales navegadores actuales, asegurando estabilidad y rendimiento en diferentes plataformas.
